
# Note Sharing Service
## Features

- Authentication
- Note Sharing 
- Rate Limiting Done with Hard Throttling

## Tech Stack
- Spring Boot
- MongoDb
- For rate limiting , i have used Guava RateLimiter

Note- Currently the DB is open to all. Everything is tested and working.
